# Jacobian-Free Unrolling (JFU) for 3D SPECT Reconstruction

This repository provides the public demo code accompanying the paper **Jacobian-Free Unrolling: memory efficient unrolled network for 3D SPECT Reconstruction**. 

## Repository Contents

This repository includes a demonstration of the JFU reconstruction framework and the associated simulated patient data. 

The following reconstruction methods are provided:

- OSEM
- MAPEM-OSL (using an RDP prior computed from the attenuation map)
- U-MAPEM (6 iterations)
- JFU-MAPEM (6 iterations)
- JFU-MAPEM (12 iterations)
- JFU-MAPEM (18 iterations)
- U-BSREM (6 iterations)
- JFU-BSREM (6 iterations)
- JFU-BSREM (12 iterations)
- JFU-BSREM (18 iterations)

## Data Availability

The complete simulated dataset used for training and evaluation will be released once all GDPR (RGPD) compliance requirements have been fulfilled.
